# Setup Guide

The full setup guide is in [README.md](README.md).

It covers all three steps:
1. **Supabase** — create database, run schema, create login user, copy credentials
2. **Vercel** — import repo, add 2 env vars, deploy
3. **OpenClaw** — paste `openclaw/OPENCLAW_SETUP.txt` into your TUI, fill in credentials, run once to verify

Total time: ~15 minutes.
